Creating tabs using Js

I made a code in javascript enables the functionality of switching between different tabs. This allows users to view different sets of content based on the tab they select. First: the code in js starts by selecting the tabs and content

elements using querySelectorAll, and then converting them into arrays using Array.from(). The code then adds a click event listener to each tab using a forEach loop. When a tab is clicked, the event listener triggers a function that removes the "active" class from all tabs, adds the "active" class to the clicked tab, and hides all content
elements. In the HTML section, there is just list (ul) with class names tabs. It contains items (li) In the Css section, The tabs are displayed as a horizontal list using flexbox, and each tab has a background color that changes on hover or when it's active.
